# 🌌 Astra OS

Astra OS is an immersive web-based os environment inspired by my love for the cosmos and space. It features a responsive interface with draggable windows, custom styling, and live external API integrations.

---

## 🚀 Key Features

*   **📺 Desktop UI Dashboard:**
    *   Responsive shortcut icons and top navigation bar.
    *   Draggable, minimizable, and maximizable window management.
*   **📡 Mission Tracker:**
    *   Fetches real-time space launch schedules from **The Space Devs API**.
    *   Interactive lists showing upcoming launches with telemetry info.
*   **🪐 Star Map Explorer:**
    *   Interactive 3D-like solar system canvas simulation.
    *   Selectable celestial bodies (planets).
    *   Real-time animations.
*   **💻 Interactive Terminal Emulator:**
    *   Full command-line experience.
    *   Custom commands: `help`, `clear`, `matrix`, `hack`, `exit`.
*   **🖼️ NASA APOD Integration:**
    *   Requests the Astronomy Picture of the Day from the NASA API.
    *   Includes an info panel displaying title, date, description, and copyright metadata.
