# 🌌 Astra OS — Sci-Fi Web Desktop

Astra OS is a premium, highly immersive browser-based desktop environment inspired by futuristic science fiction computer terminals and spacecraft telemetry interfaces. It features a rich, responsive interface with draggable/resizable windows, custom styling, and live external API integrations.

---

## 🚀 Key Features

*   **📺 Desktop UI Dashboard:**
    *   Responsive shortcut icons and top navigation bar.
    *   Draggable, minimizable, and maximizable window management.
    *   Vibrant system clock and interactive "Applications" dropdown list.
    *   Dynamic particle space-dust canvas background.
*   **📡 Mission Tracker App:**
    *   Fetches real-time space launch schedules from **The Space Devs API**.
    *   Robust fallback system (redirecting to dev endpoints if production is rate-limited).
    *   Interactive lists showing upcoming launches with telemetry info.
*   **🪐 Star Map Explorer:**
    *   Interactive 3D-like solar system canvas simulation.
    *   Selectable celestial bodies (planets) displaying orbital parameters and scan metrics.
    *   Real-time scanning animations.
*   **💻 Interactive Terminal Emulator:**
    *   Full retro command-line experience.
    *   Custom commands: `help`, `clear`, `matrix` (falling green code effect), `hack` (simulated override), `exit`.
*   **🖼️ NASA APOD Integration:**
    *   Dynamically requests the Astronomy Picture of the Day from the NASA API.
    *   Includes a slide-out info panel displaying title, date, description, and copyright metadata.
